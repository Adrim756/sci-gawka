co to CSS? – CSS to język stylów służący do wyglądu strony (kolory, układ, czcionki)

SKŁADNIA

selektor { właściwość: wartość; } – podstawowa budowa CSS

SELEKTORY

p { } – wszystkie paragrafy

.klasa { } – elementy z klasą

#id { } – element z id

* { } – wszystkie elementy

div p { } – wszystkie p w div

div > p { } – tylko bezpośrednie p w div

h1 + p { } – pierwszy p po h1

h1 ~ p { } – wszystkie p po h1

PSEUDO-KLASY

:hover – najechanie

:focus – kliknięcie inputa

:active – kliknięty element

:visited – odwiedzony link

:link – zwykły link

:target – element z #id z URL

:checked – zaznaczony checkbox / radio

KOLORY I TŁO

color: red; – kolor tekstu

background-color: blue; – kolor tła

background-image: url(img.jpg); – obraz jako tło

background-size: cover; – dopasowanie tła

TEKST

font-size: 16px; – rozmiar tekstu

font-family: Arial; – czcionka

font-weight: bold; – pogrubienie

text-align: center; – wyrównanie

text-decoration: underline; – podkreślenie

BOX MODEL

width: 100px; – szerokość

height: 100px; – wysokość

margin: 10px; – margines zewnętrzny

padding: 10px; – odstęp wewnętrzny

border: 1px solid black; – obramowanie

box-sizing: border-box; – padding w środku rozmiaru

box-shadow: 0px 0px 10px 2px #ffffff;

x — przesunięcie w poziomie (lewo/prawo)

y — przesunięcie w pionie (góra/dół)

blur — rozmycie cienia

spread — rozciągnięcie cienia (opcjonalne)

color — kolor cienia

DISPLAY

display: block; – element blokowy

display: inline; – w linii

display: inline-block; – mieszany

display: none; – ukrywa element

POSITION

position: static; – domyślne

position: relative; – względem siebie

position: absolute; – względem rodzica

position: fixed; – względem okna

top, left, right, bottom – przesunięcia

FLEXBOX

display: flex; – włącza flex

flex-direction: row; – wiersz

flex-direction: column; – kolumna

justify-content: center; – poziomo

align-items: center; – pionowo

gap: 10px; – odstępy

GRID

display: grid; – włącza grid

grid-template-columns: 1fr 1fr; – kolumny

MEDIA QUERIES

@media (max-width: 600px) – styl na telefon

Z-INDEX

z-index: 10; – kolejność warstw

OVERFLOW

overflow: hidden; – ukrywa nadmiar

overflow: scroll; – dodaje scroll

OPACITY

opacity: 0.5; – przezroczystość

CURSOR

cursor: pointer; – zmienia kursor

TRANSFORM

transform: rotate(45deg); – obrót

transform: scale(1.2); – powiększenie

TRANSITION

transition: 0.3s; – płynna zmiana

LISTY (menu)

list-style: none; – usuwa kropki

li { display: inline; } – menu poziome

LINKI

a:link – zwykły link

a:visited – odwiedzony

a:hover – po najechaniu

a:active – kliknięty

JEDNOSTKI

px – piksele

% – procent

vh / vw – ekran

em – względem rodzica (font-size)

rem – względem HTML (root)
