co to HTML? - HTML to język znaczników służący do tworzenia struktury stron internetowych.

<!-- --> - komentarz

<!DOCTYPE html>  - Informuje przeglądarkę, że to dokument HTML5

<html lang="język przykładowo (pl,en)">  - Główny element strony (wszystko jest w środku) Informuje rownież o języku przeglądarki

<head>  - Sekcja ustawień strony (tytuł, meta, style)

<title>  - Tytuł strony (widoczny na karcie w przeglądarce)

<body>  - Tu znajduje się cała widoczna zawartość strony

<p>  - Paragraf (czyli zwykły tekst)

<h1> do <h6> - Nagłówki (h1 największy, h6 najmniejszy)

<br>  - Przejście do nowej linii

<hr>  - Pozioma linia

<a href="link">  - Link do innej strony

<img src="obrazek">  - obrazek

<div> - uniwersalny blok

<header> - nagłówek strony lub sekcji

<main> - główna zawartość strony

<article> - samodzielny artykuł

<section> - sekcja tematyczna

<aside> - treści dodatkowe

<nav> - nawigacja

<footer> - stopka

<span> - to element inline, czyli „w linii” tekstu. Nie tworzy nowej linii ani bloku, służy do grupowania fragmentu tekstu lub elementów w celu stylowania CSS lub użycia w JavaScript.

<button> - Guziki (buttons), bardziej uniwersalny

<input type="button"> - guzik, prosty przycisk

<input type="submit"> - Przycisk wysyłający formularz.

<input type="reset"> - Przycisk do wyczyszczenia formularza.

<input type="text"> - Pole tekstowe, można wpisać krótką treść.

<input type="password"> - Ukrywa wpisywane znaki (kropki/gwiazdki).

<input type="email"> - Sprawdza, czy wpisany tekst wygląda jak email.

<input type="number"> - Tylko liczby w określonym zakresie.

<input type="checkbox" name="nazwa" value="wartość"> - Można zaznaczyć lub odznaczyć.

<textarea name="wiadomosc" placeholder="Twoja wiadomość"> - Duże pole tekstowe na kilka linijek.

<input type="radio" name="name"> - Można wybrać tylko jedną opcję w grupie. (ten sam podpis w name).

<meta name="description" content="strona z sciagawka"> - o meta tag w , który opisuje stronę dla wyszukiwarek i SEO. Nie jest widoczny na stronie, ale pomaga Google i innym robotom indeksować stronę.

ŁĄCZENIE SKRYPTU ORAZ STYLU W <head>

<link rel="stylesheet" href="styl.css"> - ładowanie pliku CSS ze stylem

<style> ... </style> - bezpośrednie pisanie stylów CSS w dokumencie HTML

<script src="skrypt.js"></script> - dołączenie pliku JavaScript

<script> ... </script> - bezpośrednie pisanie JavaScript w dokumencie HTML

FAV ICON

<link rel="icon" href="favicon.ico" type="image/x-icon (zamiast tego moze byc png badz jpg)"> - dodanie ikony strony (favicon)

Rozwijana lista z opcjami.

<select name="miasto">

<option value="warszawa"> warszawa </option>

</select>

LISTY.

lista uporządkowana = numerowana (ordered list)

<ol >

<li> tekst </li>

</ol>

  1. Pierwszy punkt
  2. Drugi punkt
  3. Trzeci punkt

nieuporządkowana = punktowana (unordered list)

<ul>

<li> tekst </li>

</ul>

  • Pierwszy punkt
  • Drugi punkt
  • Trzeci punkt

<tr> </tr> - (table row) – wiersz

<td> </td> - (table data) – komórka

<th> </th> - (table header) – nagłówek

Inne przydatne znaczniki tekstu

<b> tekst </b> - pogrubienie

<strong> tekst </strong> - semantycznie ważne

<i> tekst </i> - tekst pochylony

<em> tekst </em> - semantycznie podkreślone - emfaza

<u> tekst </u> - tekst podkreślony

<mark> tekst </mark> - zaznaczenie

<small> tekst </small> - mały tekst

Najważniejsze atrybuty HTML

<div class="class name"> - Nadaje klasę (do stylowania w CSS lub użycia w JS) Można używać wiele razy na stronie

<div id="id name"> - Unikalny identyfikator (tylko jeden taki na stronie) Używany do CSS, JS lub linków (#header)

<p style="styl danego elementu (tutaj akurat paragraf)"> - Styl bezpośrednio w HTML (CSS inline)

<p title="tekst wyświetlany po najechaniu"> - Pokazuje podpowiedź po najechaniu

<p hidden> - Ukrywa element

<a href="link"> - Adres strony

<a href="link" target="_blank"> - target="_blank" otwiera link w innej karcie

<img src="obraz.png"> - Ścieżka do obrazka

<img src="obraz.png" alt="opis dla obrazu"> - Opis (ważny dla dostępności i SEO)

<input type="????"> - type typ pola

  • text
  • password
  • email
  • number
  • checkbox
  • radio
  • submit

<input type="??" name="login" placeholder="Login "required>

  • name – nazwa pola
  • value – wartość pola
  • placeholder – podpowiedź
  • required – pole wymagane

<html lang="pl"> - lang język strony

<meta charset="UTF-8"> - charset kodowanie znaków

FORMULARZ ORAZ ATRYBUTY

<form action="/send" method="post/get">

action – gdzie wysłać dane

method – sposób wysyłania (GET – pobieranie danych (dane w URL), POST – wysyłanie danych (dane w body (bezpieczniejsze))

<label> - łączy opis z polem formularza

<for> - łączy z id

ENCJE HTML

  • &lt; = <
  • &gt; = >
  • &amp; = &
  • &nbsp; = spacja twarda
