SCIĄGAWKA HTML

<!DOCTYPE& html>  - Informuje przeglądarkę, że to dokument HTML5

<html lang="język przykładowo (pl,en)">  - Główny element strony (wszystko jest w środku) Informuje rownież o języku przeglądarki

<head>  - Sekcja ustawień strony (tytuł, meta, style)

<title>  - Tytuł strony (widoczny na karcie w przeglądarce)

<body>  - Tu znajduje się cała widoczna zawartość strony

<p>  - Paragraf (czyli zwykły tekst)

<h1> do <h6> - Nagłówki (h1 największy, h6 najmniejszy)

<br>  - Przejście do nowej linii

<hr>  - Pozioma linia

<a href="link">  - Link do innej strony

<img src="obrazek">  - obrazek

<div> - uniwersalny blok

<header> - nagłówek strony lub sekcji

<main> - główna zawartość strony

<article> - samodzielny artykuł

<section> - sekcja tematyczna

<asside> - treści dodatkowe

<nav> - nawigacja

<footer> - stopka

<button> - Guziki (buttons)

<input type="button"> - guzik

<input type="submit"> - Przycisk wysyłający formularz.

<input type="reset"> - Przycisk do wyczyszczenia formularza.

<input type="text"> - Pole tekstowe, można wpisać krótką treść.

<input type="password"> - Ukrywa wpisywane znaki (kropki/gwiazdki).

<input type="email"> - Sprawdza, czy wpisany tekst wygląda jak email.

<input type="button"> - guzik

<input type="number"> - Tylko liczby w określonym zakresie.

<input type="checkbox" name="nazwa" value="wartość"> - Można zaznaczyć lub odznaczyć.

<input type="radio">- Można wybrać tylko jedną opcję w grupie.
